# Anshu Pathak Portfolio

A modern, responsive portfolio site built with Astro. It includes light/dark mode support and dedicated pages for Projects, About, and Contact.

## Pages
- Home (`/`)
- Projects (`/projects`)
- About (`/about`)
- Contact (`/contact`)

## Tech Stack
- Astro
- React (theme toggle)
- CSS (custom properties + responsive layout)

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Content Updates
- Project data lives in `src/data/projects.json`.
- Global styles are in `public/styles/main.css`.
